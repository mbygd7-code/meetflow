# 킨더보드 용어집

## 서비스 용어
- **누리과정**: 국가 수준의 유아교육과정 (만 3~5세). 5개 영역: 신체운동건강, 의사소통, 사회관계, 예술경험, 자연탐구
- **월안/주안/일안**: 월별/주별/일별 교육 계획서
- **놀이계획**: 교사가 작성하는 유아 놀이 활동 계획
- **발달 기록**: 아동의 성장과 발달을 사진/텍스트로 기록한 문서
- **성장 스냅샷**: 학부모에게 공유되는 아동 활동 요약 리포트
- **알림장**: 일일 아동 활동/식단/건강 상태 학부모 전달 문서

## 기술 용어
- **안면 비식별화**: 아동 사진의 얼굴을 AI로 자동 블러/모자이크 처리
- **RAG**: Retrieval-Augmented Generation, 관련 문서를 검색해 AI 응답 품질 향상
- **Edge Function**: Supabase의 서버리스 함수 (Deno 런타임)
- **Realtime**: Supabase의 실시간 데이터 동기화 기능

## 비즈니스 용어
- **B2B SaaS**: 기업 대상 구독형 소프트웨어
- **MAU**: Monthly Active Users (월간 활성 사용자)
- **CAC**: Customer Acquisition Cost (고객 획득 비용)
- **LTV**: Lifetime Value (고객 생애 가치)
- **MRR**: Monthly Recurring Revenue (월간 반복 매출)
