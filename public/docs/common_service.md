# 킨더보드 서비스 개요

## 서비스 소개
킨더보드(KinderBoard)는 유치원·어린이집 교사를 위한 올인원 보육 관리 플랫폼입니다.

## 핵심 기능
- **아동 기록 관리**: 사진 기반 자동 발달 기록 생성 (AI 안면 비식별화 포함)
- **놀이계획 편집**: 누리과정 기반 월안/주안/일안 편집 도구
- **학부모 소통**: 성장 스냅샷, 일일 리포트, 알림장 자동 생성
- **발달 평가**: 5개 영역 레이더 차트, 타임라인 시각화
- **원 운영**: 출석, 식단, 차량, 교직원 관리

## 대상 사용자
- 유치원/어린이집 교사 (메인 사용자)
- 원장/관리자
- 학부모 (학부모 앱)

## 기술 스택
- 모바일: React Native (iOS/Android)
- 웹 관리자: React + Vite
- 백엔드: Supabase (PostgreSQL + Realtime + Edge Functions)
- AI: Claude API (회의 지원), 안면 비식별화 자체 모델

## 사업 모델
- B2B SaaS (원 단위 구독)
- 기본 플랜 / 프로 플랜 / 엔터프라이즈
